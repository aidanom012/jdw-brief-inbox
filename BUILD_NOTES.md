# JDW tactile mono spacing pass

This update keeps the swipe/desktop workflow but softens the UI:

- More breathing room on desktop and artist-folder views.
- Folder icons/cards are larger and less cramped.
- Subtle black-and-white dither texture on windows/panels.
- Salmon accent appears only on hover/destructive/action states.
- Artist folders no longer get crossed out when all campaigns are completed.
- Completed campaigns use a clean COMPLETED stamp instead of strike-through.
- Login cookie length increased to 400 days.
- Login form includes a hidden username field so browser/password managers can save the passcode more reliably.

No Supabase migration required.
